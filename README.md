## Description

Trying to deploy the fastest I can using PostgresSQL, NextJs, NestJs, Node (P3N Stack)

## Setup & Installation

### Setup

You can create a new database using docker compose just by running the following command.

```bash
$ docker compose up db
```

### Installation

You need to install all dependencies with:

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
